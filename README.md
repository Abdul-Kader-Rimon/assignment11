**BloodCare** is a user-friendly Blood Donation Application built using the **MERN stack (MongoDB, Express.js, React.js, Node.js)**. This platform connects donors with those in need of blood, enabling seamless donation management for donors, volunteers, and admins.

---

##  Project Objective
The goal of BloodCare is to streamline blood donation processes, allowing users to:

- Register as a donor, volunteer, or admin
- Create and manage blood donation requests
- Search for donors by blood group and location
- Manage user roles and permissions (Admin, Donor, Volunteer)
- Facilitate secure online donations through Stripe
- Provide a responsive and intuitive dashboard experience

---

##   Key Features

### Public
- User registration & login
- Blood donation search by blood group, district, and upazila
- View pending donation requests

### Donor Dashboard
- Profile management (view & update)
- Create new donation requests
- View, edit, or delete personal donation requests
- Track the status of donations (pending, in-progress, done, canceled)

### Admin Dashboard
- Manage all users (block/unblock, assign roles)
- Manage all donation requests
- View statistics (total users, total donations, total funding)
- filtering for all tables

### Volunteer Dashboard
- View all donation requests
- Update donation status
- Limited permissions compared to admin

### Funding
- Secure online donations via Stripe
- Track total funds and donation history
